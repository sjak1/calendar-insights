🔄 Detailed Request Flow

📝 Step-by-Step Process:

  1. User Input: User types query in web UI or sends API request
  2. FastAPI Routing: Request hits /process_query endpoint
  3. Lambda Handler: Mangum adapts ASGI to Lambda event format
  4. Query Processing: handle_query() receives query + headers
  5. AI Analysis: OpenAI GPT analyzes query with tool definitions
  6. Function Calling: GPT decides which tool(s) to call with parameters
  7. Tool Execution: Python functions execute (API calls, calculations)
  8. Result Aggregation: Tool outputs sent back to GPT
  9. Final Response: GPT generates natural language response
  10. Return to User: JSON response sent back through FastAPI
